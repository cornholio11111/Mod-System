## Documentation


Linking To API & Http (Found In ReplicatedScriptService "KeyHandler")
```lua
local KeyHandler = {
	APIKey = "ApiKey",
	WebHookHttp = "WebHook"
}
return KeyHandler
```

Enabling/Disabling Mod Type's (Found In ReplicatedScriptService "KeyHandler")
```lua
local ModTypes = {
	Textures = true,
	Models = true,
	Audio = true,
	Decals = true
}
return ModTypes
```
